# 🚢 Titanic Data Cleaning & Preprocessing

## 📌 Overview

This project focuses on cleaning and preprocessing the Titanic dataset to make it suitable for machine learning models. It demonstrates essential data preparation techniques used in real-world data science workflows.

---

## ⚙️ Key Steps

* Handled missing values using median and mode
* Removed irrelevant column (**Cabin**)
* Encoded categorical features (**Sex**, **Embarked**)
* Detected and removed outliers using IQR method
* Standardized numerical features (**Age**, **Fare**)

---

## 📊 Output

* Cleaned and processed dataset saved as:

cleaned_titanic.csv

* Improved data quality and consistency for model training

---

## 🛠️ Tech Stack

Python, Pandas, NumPy, Seaborn, Matplotlib, Scikit-learn
